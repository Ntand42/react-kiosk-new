# 🛒 React Kiosk App

A modern kiosk-style web application that allows users to browse products, add them to a cart, and complete purchases. The system also includes admin (super user) functionality for managing products and user balances.

---

## 🔐 Authentication

### 🔑 Login Page
Users must log in to access the kiosk. Authentication ensures that each user has a personalized experience, including access to their account balance and cart.

![Login](https://i.imgur.com/SJv7WTo.png)

## 👤 User Experience

### 🛍️ Adding Items to Cart
Users can browse available products and easily add items to their cart with a single action.

![Add to Cart](https://i.imgur.com/QclGtvV.png)

---

### 🧺 Viewing Cart Items
The cart provides a clear overview of selected items, allowing users to review their choices before proceeding to checkout.

![Cart Items](https://i.imgur.com/hW9NorF.png)

## 🔐 Admin (Super User) Features

### 📊 Admin Dashboard
Super users have access to a dashboard where they can manage the system, monitor activity, and perform administrative tasks.

![Admin Dashboard](https://i.imgur.com/tnNVVTZ.png)

---

### ➕ Add New Product
SuperUser can add new products to the system, making them instantly available for users to purchase.

![Add Product](https://i.imgur.com/z3D81gB.png)

## 📦 Available Scripts

### `npm start`
Runs the app in development mode.  
Open http://localhost:3000 to view it in your browser.

### `npm test`
Launches the test runner in interactive watch mode.

### `npm run build`
Builds the app for production to the `build` folder.

### `npm run eject`
⚠️ This is a one-way operation. Once you eject, you can't go back!
